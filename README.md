# LocalAI GUI (Unofficial Windows Launcher)

This project is a graphical user interface (GUI) wrapper for the popular **LocalAI** engine, rebuilt into a user-friendly Windows application (.exe) with a clean button interface.

## What does this version do?
The original LocalAI is managed via the command line and Docker. This build introduces a **convenient graphical interface**:
* Start and stop the AI engine with a single button click.
* Easily manage settings without manually editing configuration files.
* Quick access to logs directly from the application window.

---

## About the Original LocalAI Engine

**LocalAI** is the open-source AI engine. It allows you to run any model — LLMs, vision, voice, image, video — on any hardware, with no GPU required.

* **Drop-in API compatibility** — Full replacement for OpenAI, Anthropic, and ElevenLabs APIs.
* **36+ backends** — Supports llama.cpp, vLLM, transformers, whisper, diffusers, MLX, and more.
* **Any hardware** — Optimized for NVIDIA, AMD, Intel, Apple Silicon, Vulkan, or CPU-only setups.
* **Multi-user ready** — API key authentication, user quotas, and role-based access controls.
* **Built-in AI agents** — Autonomous agents featuring tool use, RAG, MCP, and custom skills.
* **Privacy-first** — Your data never leaves your local infrastructure.
